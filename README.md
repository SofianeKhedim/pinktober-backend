# Getting started

To get the server running locally:

- clone the repo and install the dependencies :
```console
$ git clone https://github.com/fatimaBenazzou/pinktober.git
$ cd back-end
$ npm install
```
- create .env file 

- to start the local server type this command :
```console
$ npm run dev
```
